# IRON LOG — 筋トレ記録 PWA

iPhoneのホーム画面から起動できる、オフライン対応の筋トレ記録アプリです。

---

## 🚀 使い方（ローカルで動かす）

### 方法 A：VSCode Live Server（推奨）
1. VSCodeで `ironlog` フォルダを開く
2. `index.html` を右クリック →「Open with Live Server」
3. iPhoneでPC側のIPアドレスにアクセス（例：`http://192.168.1.10:5500`）

### 方法 B：Python簡易サーバー
```bash
cd ironlog
python3 -m http.server 8000
```
→ `http://localhost:8000` にアクセス

### 方法 C：GitHub Pages（推奨・外出先でも使える）
1. GitHubに `ironlog` フォルダをリポジトリとしてpush
2. Settings → Pages → Source: main / root に設定
3. `https://ユーザー名.github.io/ironlog/` でアクセス可能

> ⚠️ PWA（Service Worker）は **https または localhost** でのみ動作します。
> GitHub Pages は自動でhttpsになるので最も簡単です。

## 📲 iPhoneのホーム画面に追加する方法

1. iPhoneのSafariでアプリのURLを開く
2. 画面下部の共有ボタン（□↑）をタップ
3.「ホーム画面に追加」を選択
4. 名前を確認して「追加」

これで**ネイティブアプリのように**使えます（オフラインでも動作）。

## 🖼 アイコンの差し替え方

`icons/` フォルダに、以下のサイズのPNG画像を用意して入れてください：

| ファイル名       | サイズ    | 用途               |
|------------------|-----------|--------------------|
| icon-180.png     | 180×180   | iPhone ホーム画面（最重要） |
| icon-167.png     | 167×167   | iPad Pro           |
| icon-152.png     | 152×152   | iPad               |
| icon-192.png     | 192×192   | Android / PWA      |
| icon-512.png     | 512×512   | スプラッシュ画面   |
| icon-32.png      | 32×32     | ブラウザタブ       |

**簡単な方法**：180×180の画像を1枚用意して、複数サイズにリサイズするだけでOKです。

## ✨ 機能一覧

### 🏋️ トレーニング画面
- 種目ごとにカードで管理
- **セット記録**：ボタンをタップするたびにカウントアップ
  - 各セットの記録時刻を表示
  - 「↩ 取消」で最後のセットを削除（誤操作対策）
- **レストタイマー**（補助機能・セット記録とは独立）
  - カウントダウン：設定秒数から0へ（60/90/120/180秒プリセット＋カスタム）
  - ストップウォッチ：上限なしで増加
- 種目カードのドラッグ＆ドロップで並び替え

### ✏️ 種目編集
- 種目の追加・削除・名前変更・重量変更

### 💾 ログ保存
- その日の種目・重量・セット数と**本日の総重量**を記録

### 📅 ログ画面
- 日付別に過去のログを一覧表示

### 📊 統計画面
- **累計扱った総重量**
- 記録日数・1日平均重量
- 直近8セッションの総重量グラフ

## 🔒 データについて
すべてのデータはブラウザの `localStorage` に保存されます。
サーバーには一切送信されず、プライバシー安全です。
アプリをアンインストール（Safariのデータを消去）するとデータも消えるのでご注意ください。
